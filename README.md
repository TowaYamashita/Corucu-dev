# Corucuとは？
学内での授業の変更や追試験・再試験についての情報を一元的に管理し、
いつでもどこでも情報の閲覧ができるWebアプリケーションです。

---

# What is "Corucu" ?
"Corucu" is a web application that allows you to view and edit information related to classroom and additional examinations.
